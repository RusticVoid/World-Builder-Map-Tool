# How to Use

Upload Map: Click “Import Map” and choose an image.

Add Marker: Double-click anywhere on the map.

Edit Marker: Click a marker, then edit its info in the sidebar.

Move Marker: Click and Drag a marker to a new spot.

Categories: Use + to add a category and − to remove one in the sidebar.

Custom Icons: Select a marker, then upload an icon in the sidebar.

Regions: Use the drawing tools on the left to draw areas.

Save/Load: "Save World" exports a JSON file. "Load World" restores it.
