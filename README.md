# How to Use

### Import Map: 
Click "Import Map" and select an image file. The image becomes your world map.

### Add Marker: 
Double-click anywhere on the map to create a new location marker.

### Edit Marker: 
Click a marker to edit its name, category, color, icon, description, and link in the sidebar.

### Move Marker: 
Click and drag any marker to reposition it.

### Categories: 
Use the + button to create categories and the − button to remove them.

### Custom Icons: 
Select a marker and upload a custom image icon.

### Draw Regions: 
Use the drawing tools on the left side of the map to create territories, kingdoms, provinces, biomes, or other areas.

### Edit Regions: 
Click a region to edit its name and color in the sidebar.

### Save World: 
Exports all markers, regions, labels, colors, categories, and settings into a JSON file.

### Load World: 
Restores a previously saved world from a JSON file.
