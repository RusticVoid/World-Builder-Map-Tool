World Builder Guide

Build your map faster

Import a map image, place locations, draw regions, customize labels, and save everything as a reusable world file.

Quick Start

  1. 1Import Map

    Choose an image to use as your world map.

  2. 2Add Places

    Double-click the map to drop markers.

  3. 3Draw Areas

    Use the drawing tools on the left to outline regions.

  4. 4Save World

    Export your map, markers, regions, and styling to JSON.

Map Files

Import Map loads a PNG, JPG, or other image as the map canvas.

Save World exports all map data into a JSON file.

Load World restores a previously saved JSON world.

Markers

Add: Double-click anywhere on the map.

Edit: Click a marker, then use the sidebar for name, category, color, icon, description, and link.

Move: Drag a marker to reposition it.

Categories & Icons

Use + to create marker categories and − to remove them.

Upload a custom marker icon, or clear it to return to a colored dot.

Regions

Use the left-side drawing tools to create territories, kingdoms, provinces, biomes, routes, or borders.

Click a region to edit its name, color, label visibility, text color, background, size, font, style, and curve.

Best Practices

  • Save before making large changes.
  • Use categories for cities, ruins, landmarks, factions, or quest locations.
  • Use region colors to separate nations, climates, territories, or political borders.
  • Add descriptions and links to turn your map into a lightweight world encyclopedia.
